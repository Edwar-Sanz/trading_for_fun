Principales variables y operadores en MQL5

Variables en MQL5

Tipos de Datos Básicos

  • int (Entero): Números enteros, como 1, 2, -3.
  • double (Doble precisión): Números de punto flotante, como 3.14, 2.71.
  • bool (Booleano): Solo puede tener valores true o false.
  • string (Cadena de texto): Almacena texto, como “Hola Mundo”.

Tipos de Datos Compuestos

  • array (Arreglo): Conjunto de valores del mismo tipo.
  • struct (Estructura): Grupo de diferentes tipos de variables bajo un mismo nombre.

3. Tipos de Datos Específicos

  • datetime: Representa fecha y hora.
  • color: Almacena colores en formato RGB.
  • enum: Define una lista de valores posibles.

Operadores en MQL5

1. Operadores Aritméticos

  • + (Suma)
  • - (Resta)
  • * (Multiplicación)
  • / (División)
  • % (Módulo)

2. Operadores Relacionales

  • == (Igualdad)
  • != (Diferente)
  • > (Mayor que)
  • < (Menor que)
  • >= (Mayor o igual que)
  • <= (Menor o igual que)

3. Operadores Lógicos

  • && (AND lógico)
  • || (OR lógico)
  • ! (Negación lógica)

4. Operadores de Asignación

  • = (Asignación)
  • +=, -=, *=, /= (Operadores de asignación compuesta)

Ejemplos

Ejemplo 1

En este ejemplo utilizamos variables de tipo int y double para asignar valores, también utilizamos el operador + para sumar dos valores.


    void OnInit()
    {
        int a = 10;
        double b = 5.5;
        double suma = a + b;
        Print("La suma de a y b es: ", suma);
    }

Ejemplo 2

En este ejemplo, utilizamos operadores relacionales y lógicos para comparar dos valores y asignar el resultado a una variable de tipo bool.

    
    void OnInit()
    {
        int x = 20;
        int y = 15;
        bool resultado = (x > y) && (y != 0);
        Print("El resultado de la comparación es: ", resultado);
    }

Ejemplo 3

En este ejemplo, se utiliza un enum para definir los tipos de ordenes posibles. Luego, se utiliza el enum en una variable de tipo OrderType y se compara con el valor BUY para determinar si la orden es de compra o venta.


    enum OrderType
    {
        BUY,
        SELL,
        PENDING
    };

    void OnInit()
    {
        OrderType order = BUY;
        if(order == BUY)
        {
            Print("Orden de compra seleccionada");
        }
    }
